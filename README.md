## Prerequisites

- ohmyzsh lol

## How 2 install

- ```cd ~/.oh-my-zsh/themes```
- ```wget https://raw.githubusercontent.com/wshloic/lenny-ohmyzsh/master/lenny.zsh-theme```
- ```vim ~/.zshrc```
- set ZSH_THEME to "lenny"
- save and quit
- ```source ~/.zshrc```
- you're done
